git --version
git --help


git init (Инициализация гита в проекте, применяем только 1 раз)

git status (Проверка и отображение состояния проекта)

git add (Команда для добавления файла в stage)
Примеры:

git add <имя вашего файла> (добавляет ваш файл)
git add -A (добавляет все файлы на всех уровнях вложенности)
git add --all (Аналогична верхней команде)
git add . (добавляет все файлы на текущем и всех нижних уровнях вложенности)

git status

git config
git config [--global | --local] user.name 'Ваше имя' (устанавливает новое имя)
git config [--global | --local] user.email 'Ваша почта' (устанавливает новую почту)

git config --list (выводит все данные с файла конфигураций)

git  (добавляет ваш коммит в историю и ставит HEAD на этот коммит)

.gitignore

git branch (показывает набор веток и текущую ветку)
git branch <имя ветки> (создаёт новую ветку)
git branch -D <имя ветки> (удаляет указанную ветку)

git checkout <имя ветки> (переходит на существующую ветку)

git checkout -b <имя ветки> (создаёт новую ветку и переходит на неё)

git merge <имя ветки> (соединяет/сливает текущую ветку с указанной)

readme.md

git log (история коммитов)

git clone (копирование существующего удаленного репозитория)

git pull (обновление данных с удаленного репозитория в локальный)
